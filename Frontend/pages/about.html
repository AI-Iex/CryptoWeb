Backend Overview

The backend is built with Python and FastAPI, following a clean, modular architecture:

  • FastAPI for high-performance REST endpoints.
  • SQLAlchemy as ORM to interact with a PostgreSQL database.
  • Routes for organizing and exposing API endpoints.
  • Auth Public Endpoints for sign-up and login, with protection against unauthorized admin-role assignment.
  • Services to encapsulate business logic and process requests.
  • Repositories to handle CRUD operations cleanly.
  • Schemas to define DTOs and avoid exposing sensitive fields.
  • News API integration handled server-side via a private route to protect the API key and control query limits.
  • Chatbot exposed through a protected route requiring user authentication.
  • Docker for containerization, local development, and deployment.
  • JWT for secure token-based authentication.
